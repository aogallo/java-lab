# Ejemplo tutor - Primer proyecto

## Objetivo

Mostrar como crear o abrir un proyecto Maven en NetBeans y ejecutar `Main.java`.

## Pasos

1. Abrir NetBeans.
2. Crear un proyecto `Java with Maven > Java Application`.
3. Ubicar `pom.xml`.
4. Abrir `src/main/java`.
5. Crear o revisar `Main.java`.
6. Ejecutar el proyecto.

## Codigo del tutor

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Sistema Academico Java");
    }
}
```
