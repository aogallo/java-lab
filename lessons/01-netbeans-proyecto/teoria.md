# Teoria - Crear el proyecto en NetBeans

## Definiciones

- **Proyecto**: carpeta organizada con codigo, configuracion y recursos.
- **Maven**: herramienta que organiza el proyecto Java y usa `pom.xml`.
- **`pom.xml`**: archivo donde se define el proyecto, dependencias y plugins.
- **`Main.java`**: clase inicial desde donde arranca el programa.

## Idea clave

Antes de aprender POO o base de datos, el alumno debe saber crear, abrir y ejecutar el proyecto. Si no puede ejecutar, no puede comprobar lo que aprende.
