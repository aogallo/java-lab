# Teoria - Menu por consola

## Definiciones

- **Variable**: espacio para guardar un valor.
- **Condicional**: permite decidir que camino tomar.
- **Ciclo**: repite instrucciones.
- **Metodo**: bloque de codigo con nombre.
- **Scanner**: clase de Java para leer entrada por consola.

## Idea clave

El sistema academico empieza como un menu. Ese menu sera la puerta para crear estudiantes, cursos, inscripciones y notas.
