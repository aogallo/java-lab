# Teoria - Clases y objetos con Student

## Definiciones

- **Clase**: molde para crear objetos.
- **Objeto**: instancia concreta de una clase.
- **Atributo**: dato que pertenece al objeto.
- **Constructor**: metodo especial para crear objetos con valores iniciales.

## Idea clave

El sistema academico necesita representar estudiantes. Esa representacion empieza con una clase `Student`.
