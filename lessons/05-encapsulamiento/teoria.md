# Teoria - Encapsulamiento

## Definiciones

- **Encapsulamiento**: proteger atributos y acceder a ellos mediante metodos.
- **Getter**: metodo que lee un atributo privado.
- **Setter**: metodo que modifica un atributo privado.

## Idea clave

No usamos Lombok. El alumno debe escribir getters y setters para entender como se protege el estado del objeto.
