# Teoria - Herencia con Person, Student y Teacher

## Definiciones

- **Herencia**: permite crear una clase a partir de otra.
- **Clase padre**: clase con datos comunes. En este caso `Person`.
- **Clase hija**: clase que hereda datos comunes y agrega datos propios.
- **`extends`**: palabra clave usada para heredar en Java.

## Idea clave

`Student` y `Teacher` comparten datos: nombre, apellido, email. No conviene repetir eso dos veces. Creamos `Person` para representar lo comun.
