# Teoria - MySQL Workbench

## Definiciones

- **Base de datos**: lugar donde se guardan datos.
- **Tabla**: estructura con filas y columnas.
- **Clave primaria**: identificador unico.
- **Clave foranea**: relacion entre tablas.
- **Script SQL**: archivo con comandos SQL.

## Idea clave

La base de datos se crea manualmente en Workbench para entender que ocurre antes de conectar Java.
