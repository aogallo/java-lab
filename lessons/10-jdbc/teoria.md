# Teoria - JDBC

## Definiciones

- **JDBC**: API de Java para conectarse a bases de datos.
- **Connection**: conexion abierta con la base de datos.
- **PreparedStatement**: consulta preparada con parametros.
- **ResultSet**: resultado de una consulta.

## Idea clave

JDBC conecta el repository con MySQL. Los scripts se ejecutan manualmente antes en Workbench.
