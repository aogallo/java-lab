# Lecciones del Sistema Academico Java

Estas lecciones construyen un solo proyecto progresivo: un sistema academico por consola.

## Como usar este material

El tutor usa este repo para explicar, guiar y revisar. El alumno puede crear su propio repositorio desde cero y avanzar siguiendo cada practica.

## Estructura de cada leccion

- `teoria.md`: definiciones y conceptos.
- `ejemplo-tutor.md`: ejemplo que desarrolla el tutor.
- `practica-alumno.md`: implementacion que hace el alumno.
- `reto.md`: tarea para reforzar.

## Lecciones

1. `01-netbeans-proyecto`: crear y ejecutar el proyecto.
2. `02-git-basico`: versionar el avance del alumno.
3. `03-menu-consola`: construir el primer menu.
4. `04-clases-objetos-student`: crear `Student`.
5. `05-encapsulamiento`: aplicar getters y setters.
6. `06-herencia-person-student-teacher`: crear `Person`, `Student` y `Teacher`.
7. `07-colecciones-repository`: guardar estudiantes en memoria.
8. `08-arquitectura-capas`: separar responsabilidades.
9. `09-mysql-workbench`: crear scripts SQL manuales.
10. `10-jdbc`: conectar Java con MySQL.
11. `11-integracion-final`: unir el sistema.
